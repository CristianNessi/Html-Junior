Body.

una vez definida la parte del body, comenzamos a impletar etiquetas basicas para que nuestro sitio web empiece a tomar forma. En este aparte vamos a definir varias etiquetas como "Titulo", "Subtitulos", "Parrafos", "Tablas", "Listas", "Section", "Divisores", "Articulos", "Clases" Una web se esta basada en bloques. Y cada bloque utiliza un espacio determinado en el body. Hay etiquetas que ultizan todo el ancho del bloque y otras que ultilizan la mitad del bloque. Que quiere decir ? Que si ponemos una etiqueta que utiliza el ancho completo del bloque la proxima etiqueta va a saltar hacia abajo para continuar. En cambio, una etiqueta que no utiliza el ancho completo, la proxima etiqueta va a continuar al lado y no va a saltar a la proxima linea desperdicuando espacio.

Estas son algunas de las mas comunes.

Etiqueta "section".

El elemento de HTML (section) representa una sección genérica independiente de un documento, que no tiene un elemento semántico más específico para representarla. Las secciones siempre deben tener un título. Esta etiqueta no ocupa el ancho total por defecto, se ajusta al ancho de su contenedor principal. El ancho de la sección dependerá de las propiedades CSS aplicadas a ella o a sus contenedores. Tambien funciona como contenedor, el cual puede tener varias etiquetas (article).

Etiqueta "article".

La etiqueta (article) en HTML se utiliza para definir una sección de contenido independiente y autónoma en un documento. Es decir, el contenido dentro de la etiqueta (article) debería ser capaz de funcionar por sí mismo y tener un significado o relevancia semántica incluso si se extrae y se presenta de manera aislada.

Etiqueta "class".

Como si nombre lo indicada una etiqueta (class) se utliza para agrupar un conjunto de datos. Su funcion es agruparlos para luego poder llamarlos cuando necesitemos el dato. La etiqueta (class) se utiliza mucho para combinarla con "CSS" y "JavaScript". Se le puede dar clases a cualquiera elemento.

A difencia de la otra etiqueta (ID) como su nombre lo indica su funcion es identificar un unico elemento.

Etiqueta "div".

La etiqueta (div) en HTML es un elemento de bloque que se utiliza para crear divisiones o contenedores genéricos en una página web. Es un contenedor sin significado semántico propio. No tiene un propósito específico o contenido predeterminado, por lo que su función principal es actuar como un bloque de construcción flexible.
